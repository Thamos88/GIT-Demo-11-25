GIT rockt!

git reset entfernt Commits aus dem aktuellen Branch, verschiebt den HEAD und kann den Index oder die Arbeitskopie ändern (soft, mixed, hard). Vorsicht: Datenverlust möglich; bei falscher Nutzung sehr vorsichtig.

Branches ermöglichen parallele Entwicklung, isolieren Features, vereinfachen Zusammenarbeit, erlauben Experimentieren, erleichtern Codeintegration und Versionsverwaltung ohne Hauptzweig zu stören, und Rückverfolgbarkeit.
